## Task history

To view your full task history you can click on then **View task history** button above the task list.

<Frame>
  <img
    src="/images/view-task-history.png"
    alt="View task history button above the task list"
    style={{ borderRadius: '0.5rem' }}
  />
</Frame>

From here you will be able to view a calendar chart showing all days you used forget.work and worked on tasks, with some information for each day.

<Frame>
  <img
    src="/images/task-history-chart.png"
    alt="Calendar chart showing task activity by day"
    style={{ borderRadius: '0.5rem' }}
  />
</Frame>

You can hover over each day to view some high level stats and filter by year.

<Frame>
  <img
    src="/images/task-history-hover.png"
    alt="Hover tooltip showing high level task stats for a day"
    style={{ borderRadius: '0.5rem' }}
  />
</Frame>

You can click on any day either in the chart or the list and it will take you to a page which displays analytics about your tasks on that day with a full list of them.

<Frame>
  <img
    src="/images/task-history-day.png"
    alt="Task history day analytics with a full list of tasks"
    style={{ borderRadius: '0.5rem' }}
  />
</Frame>
